openwrt-wrt1x00ac-patchs
=================================================


简述:
    这是我购买WRT1200AC后，为其编译基于OpenWrt的固
    件时，对源码所做更改的补丁集，会陆续把补丁放上
    来。打补丁时从OpenWrt源码的根目录开始打即可。

文件概述:
    [1] marvell_switch.patch - 针对marvell交换机在
            luci的交换机页面添加端口状态显示。


测试环境:
    在 OpenWRT 15.05.1 上开发


依赖关系:
    无
